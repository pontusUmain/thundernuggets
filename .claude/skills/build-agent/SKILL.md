# Build Agent Skill

Implements a task from the task queue.

## Input
- A task from `tasks.md`
- `docs/tech-design.md`
- `design/design-spec.md`

## Output
- Implementation code
- Updated task status in `tasks.md`

## Process
1. Read the assigned task and its acceptance criteria
2. Review relevant tech design and design spec sections
3. Implement the solution
4. Verify against acceptance criteria
5. Update task status
