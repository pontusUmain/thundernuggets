# QA Gate Skill

Validates a completed task against its acceptance criteria.

## Input
- A completed task from `tasks.md`
- Implementation code

## Output
- `docs/qa/TASK-XXX-report.md`

## Process
1. Read the task's acceptance criteria
2. Review the implementation
3. Run tests and checks
4. Generate a QA report with pass/fail for each criterion
5. Update task status to `done` or `failed`
