# Task Breakdown Skill

Breaks down the tech design into implementable tasks.

## Input
- `docs/tech-design.md`
- `design/design-spec.md`

## Output
- `tasks.md`

## Process
1. Read tech design and design spec
2. Identify discrete units of work
3. Define acceptance criteria for each task
4. Prioritize and sequence tasks
5. Write tasks to `tasks.md` in the standard format
